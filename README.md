# 테트리스 (교육용)

HTML, CSS, JavaScript만 사용하는 브라우저 테트리스 프로젝트입니다.

## 실행 방법

1. 이 프로젝트 폴더를 연다.
2. `index.html` 파일을 더블클릭하거나, 브라우저로 드래그하여 연다.
3. **시작** 버튼을 눌러 게임을 시작한다.

### VS Code Live Server 사용 (선택)

VS Code의 **Live Server** 확장을 설치한 경우:

1. `index.html`을 우클릭한다.
2. **Open with Live Server**를 선택한다.

## 조작법

| 키 | 동작 |
|----|------|
| `←` (`ArrowLeft`) | 왼쪽으로 한 칸 이동 |
| `→` (`ArrowRight`) | 오른쪽으로 한 칸 이동 |
| `↓` (`ArrowDown`) | 한 칸 빠르게 내리기 (소프트 드롭) |
| `↑` (`ArrowUp`) | 블록 시계 방향 90° 회전 |
| `Space` | 즉시 바닥까지 내리기 (hard drop) |

모든 조작은 `canMove` 충돌 판정을 통과할 때만 적용됩니다. 회전 후 벽이나 고정 블록과 겹치면 회전은 취소됩니다.

## 파일 구성

| 파일 | 설명 |
|------|------|
| `index.html` | 게임 화면 구조 및 조작법 안내 |
| `style.css` | 레이아웃 및 스타일 |
| `script.js` | 게임 로직, 입력 처리, 렌더링 |

## 현재 상태

- 10×20 게임 보드
- 7가지 테트로미노 생성 및 자동 낙하
- 충돌 판정 (`canMove`) 및 블록 고정
- 키보드 조작: 좌우 이동, 소프트 드롭, 회전, hard drop
- 점수 UI (초기값 0, 라인 삭제 점수는 미구현)
- 시작/재시작 버튼
